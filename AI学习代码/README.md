## 🚀 如何使用

### 1. 运行示例代码

示例代码是完整可运行的，展示了各种机器学习技术的标准用法：

```bash
# 进入示例代码目录
cd 示例代码

# 运行示例
python 01_线性回归基础.py
python 02_数据集划分.py
# ... 其他示例
```

### 2. 完成练习

练习代码中留有 `TODO` 注释，需要你根据提示完成代码：

```bash
# 进入练习代码目录
cd 练习代码

# 打开练习文件，完成 TODO 部分
python 练习_01_线性回归.py
```

### 3. 查看答案

如果你遇到困难，可以参考对应的答案文件：

```bash
python 答案_01_线性回归.py
```

## 📚 学习内容

### 示例代码

1. **线性回归基础** - 学习如何使用scikit-learn进行线性回归分析
   - 数据生成和可视化
   - 模型训练和评估
   - 多变量回归和多项式回归

2. **数据集划分** - 掌握训练集、验证集、测试集的划分方法
   - 不同划分比例的影响
   - 分层抽样技术
   - 交叉验证实现

3. **AI模型训练** - 完整的模型训练流程
   - 数据加载和预处理
   - 特征标准化
   - 模型训练和评估
   - 结果可视化

4. **分类算法** - 多种分类算法的比较和使用
   - 逻辑回归、决策树、随机森林、SVM、KNN
   - 模型性能比较
   - 特征重要性分析

5. **聚类算法** - 无监督学习的聚类技术
   - K-Means算法
   - 最佳K值选择（肘部法则、轮廓系数）
   - 异常检测应用

## 💡 常见问题

**Q: 运行代码时出现中文乱码？**  
A: 确保你的IDE或终端支持UTF-8编码，或者在代码中设置中文字体。

**Q: 图形界面不显示？**  
A: 确保你的环境支持图形显示。如果在远程服务器上运行，可能需要使用`Agg`后端保存图片而不是显示。

**Q: 如何保存生成的图表？**  
A: 在`plt.show()`之前添加 `plt.savefig('figure.png')` 即可保存。
